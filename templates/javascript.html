#  Strings

In JavaScript, strings are sequences of characters enclosed within either single quotes ' or double quotes ". Strings must begin and end with quotation marks .

Example

  let   ' Hi Panda '
  let   " Hi Panda "

The quotes are there to indicate the enclosed text is a value, not code.

#  Variables

A variable is a container for a value, like a number we might use in a sum, or a string that we might use as part of a sentence.

Basic rules to declare a variable in JavaScript:

  • These are case-sensitive.
  • Can only begin with a letter, underscore _ or $ symbol.
  • It can contain letters, numbers, underscore, or $ symbol.
  • A variable name cannot be a reserved keyword.

JavaScript is a dynamically typed language so the type of variables is decided at runtime. Therefore there is no need to explicitly define the type of a variable. We can declare variables in JavaScript in three ways:

  • Using let keyword
  • using var keyword
  • using const keyword

Example

  let   $ = "My Panda"
  var   a = "Hi Panda"
  const   _panda = "bamboo"

#  Numbers

The Number constructor contains constants and methods for working with numbers. Values of other types can be converted to numbers using the Number() function.

If a number does not contain a decimal, it's referred to as an integer. JavaScript has only one type of number. Numbers can be written with or without decimals.

Example

  let   A = 2.41    // A number with decimals
  let   P = 8    // A number without decimals

#  Operators

JavaScript Operators are symbols used to perform specific mathematical, comparison, assignment, and logical computations on operands.
They are fundamental elements in JavaScript programming, allowing developers to manipulate data and control program flow efficiently.

Operators are the symbols between values that allow different operations like addition + , subtraction - , multiplication * , division / , modulus % , and exponentiation ** .

Example :

1. Addition operator(+)

  let   x = 2;
  let   y = 3;
  let   z = x + y;

2. Substractor operator(-)

  let   a = 7
  let   b = 4;
  let   z = x - y;

3. Multiplication operator(*)

  let   p = 4;
  let   q = 3;
  let   z = x * y;

4. Division operator(/)

  let   x = 8;
  let   y = 4;
  let   z = x / y;

5. Modulus operator(%)

  let   m = 4;
  let   n = 2;
  let   z = x % y;

5. Exponentiation operator(**)

  let   x = 3;
  let   y = 2;
  let   z = x ** y;

#  Functions

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

A function definition (also called a function declaration, or function statement) consists of the function keyword, followed by:

  • The name of the function.
  • A list of parameters to the function, enclosed in parentheses and separated by commas.
  • The JavaScript statements that define the function, enclosed in curly braces, { /* … */ }.

Example

    function square(number) {
        return number * number;
   }

Using the return keyword is very important. When you encounter a function, you must return your answer, otherwise the code won't work.

#  Arrays

An array in JavaScript is a data structure used to store multiple values in a single variable. It can hold various data types and allows for dynamic resizing.

We can write array like:

    const cars = ["Panda", "Bamboo", "Eat"];

#  Mutability

Elements in arrays are mutable, which means they can be changed. For example, let's say we've got an array of numbers:

    const cars = ["Panda", "Bamboo", "Eat"];
To change the value of "Eat" (at index 2), we do this:
    cars[3] = "Animal"

#  .length

Javascript has a .length property that returns the size of an array as a number(integer).

Here's an example of how to use it:

    let number = [5,1,7,4]
To find length of array :
    number.length

#  Objects

Objects in JavaScript are the most important data type and form the building blocks for modern JavaScript.
Unlike primitive data types (Number, String, Boolean, null, undefined, and symbol), which hold a single value, objects can hold multiple values as properties.

We can create objects in JavaScript like:
    const games = {
        Indoor games: "Chess",
        Outdoor games: "Cricket",
        E-games: "Free Fire"
    }

#  Key-Value Pairs

An object contains properties, or key-value pairs. Each property has a name, which is also called a key, and a corresponding value.

Objects are variables too. But objects can contain many values.

An object literal is a list of name:value pairs inside curly braces {}

Example:

    const car = {type:"Fiat", model:"500", color:"white"};

#  Dot Notation

In the object.propertyName syntax, the propertyName must be a valid JavaScript identifier which can also be a reserved word.

You want to access the E-games of game, you would return E-game.game. This is called dot notation

Example:

    const car = {type:"Fiat", model:"500", color:"white"};

#  Loops

The JavaScript loops are used to iterate the piece of code using for, while, do while or for-in loops. It makes the code compact. It is mostly used in array.

There are four types of loops in JavaScript.

  • for loop
  • while loop
  • do-while loop

For loop in Java Script

A for loop in JavaScript repeatedly executes a block of code as long as a specified condition is true. It includes initialization, condition checking, and iteration steps, making it efficient for controlled, repetitive tasks.

Syntax:

    for (initialization; condition; updation){
    Code to be executed
    }

Example:

    for (let i = 0; i < 5; i++) {
    document.write(i + "< br>")
    }

while loop in Java Script

Syntax:

    while (condition){
    Code to be executed
    }
Example:
    while(i<=5) {
    document.write(i + "< br>")
    i++;
    }

do-while loop in Java Script

Syntax:

    do{
    Code to be executed
    }while(condition);
Example:
    do {
    document.write(i + "< br>")
    i++;
    }while(i<=5);
